# Website Galeri UMKM Balikpapan

Website Galeri UMKM Balikpapan merupakan platform katalog digital yang dikembangkan sebagai bagian dari kegiatan **Inovasi Sosial (INSOS) Institut Teknologi Kalimantan (ITK)**.

Website ini bertujuan untuk membantu Galeri UMKM Balikpapan dalam mempromosikan produk UMKM secara lebih efektif melalui katalog online yang dapat diakses oleh masyarakat kapan saja.

---

## Fitur Utama

- Katalog Produk UMKM
- Profil UMKM
- Pencarian Produk
- Kategori & Jenis Produk
- Dashboard Admin
  - Kelola UMKM
  - Kelola Produk
- Responsive Design

> Beberapa fitur masih dalam tahap pengembangan.
